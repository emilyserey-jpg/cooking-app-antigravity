💡 How this prototype works:

This page embeds a standard YouTube video using Google's official YouTube Player API. Because we have programmatic control, the Javascript runs a high-precision background timer. When the video goes outside the boundaries of the active step, the script instantly triggers a player.seekTo() command back to the start timestamp, achieving a 100% database-free, ad-funded loop experience.
